TOPIC
JSON

JSON is an interesting topic to tackle when you're talking both this class, Object-Oriented Development, and CIT 230: Web-Frontend Development. Both use JSON, but JSON in java isn't quite the same as JSON—JavaScript Object Notation. For my JSON, I decided to continue my theme of my using my daughter's basic info. I setup JSON to map her information to keyword labels. I think of this as being most useful for website contact forms. I can have an html form sent as an email to the webmaster using JSON. Now, if only I couls get the JavaScript Object Notion to work as well on my CIT 230 project.




TOPIC
System Test

Ah, a good ol' system test. I really enjoy this kind of exercise. I decided I could use this as a parenting moment killing two birds with one stone, so to speak. My second-born struggles to remember (or pay attention, dealer's choice on the semantics) how to properly use our toasteroven and then gets frustrated that her foos isn't prepared properly. So! This System Test is for her. I did a step-by-stem system test to test the different heating modes on our toaster, incorporating proper general usage. I integrated a rinse-repeat style test that required not only testing the different heating modes, but matching them with appropriate food items on the testing list. Once I explained this was for science, she came onboard and was a wonderful tester. She performed all the tests admirable excluding a rough start with the door. I got a serviceable systems test and now the pre-teen can feed herself a little better. WIN-WIN!
